# Favorite Animation

アニメ作品の感想を記録・共有可能サービス。

https://favoriteanimation.herokuapp.com/


## 環境

* 言語(Ruby 2.5.3)
* フレームワーク(Rails 5.2.4.3)
* 開発(Cloud9)

## 機能・技術

* 画像(carrierwave,mini_magick,cloudinary)
* ページネーション(kaminari)
* アニメーション(bootstrap-tooltip-rails,jquery-rails)
* インフラ(Heroku)
* デザイン(Bootstrap)
